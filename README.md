# 语音识别api_java

#### 项目介绍
使用百度api完成录音识别成一个关键字的集合


#### 安装教程

1. git clone
2. ./gradlew compileJava
3. 在src/main/resources/application.yml 配置自己百度api的相关参数
4. ./gradlew bootRun
5. [http://localhost:2345/file/upload](http://localhost:2345/file/upload) 进行上传文件
